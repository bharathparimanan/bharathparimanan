## 💡 About Me

I have always been fascinated by the power of data—how it's structured, processed, and leveraged for decision-making.

My experience working with **Node.js, PostgreSQL, MySQL, and RESTful APIs** has given me deep insights into **data management**, **scalability**, and **system performance**.

This curiosity led me to pursue a **Master’s in Data Science** at the **University of Hertfordshire**, where I’m deepening my expertise in **data engineering**, **big data processing**, and **machine learning**.

🎯 **My goal:** To design robust ETL pipelines, optimize data workflows, and harness cloud platforms to manage large-scale, real-time data systems.

---

## 🧠 Core Areas of Focus

- 🔹 **Data Engineering:** ETL pipeline development, database optimization, cloud-based data solutions, Apache Spark, Hadoop, and Kafka for real-time streaming.
- 🔹 **Data Science & Modeling:** Predictive analytics, classification & regression models, data visualization, survival analysis, feature engineering.
- 🔹 **Backend & Databases:** RESTful API design, SQL & NoSQL systems, database scaling, and pipeline automation.
